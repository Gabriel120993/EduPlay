# Cumplimiento COPPA / GDPR-K (menores)

Documento interno de verificación para EduPlay. No sustituye asesoría legal.

## Verificación parental

- [x] Proceso documentado en `terms-of-service.md` y `privacy-policy.md`
- [x] Alta de menores solo por tutor autenticado; aprobación explícita en panel familiar
- [ ] Método de verificación reforzado (p. ej. cargo simbólico, documento) si el producto escala fuera de UE/AR

## Datos de menores

- Solo se recolectan datos necesarios para el funcionamiento educativo y social supervisado
- No se venden datos a terceros con fines publicitarios
- Eliminación a pedido del tutor: contactar soporte o usar flujos de baja documentados en la política de privacidad

## Consentimiento en registro de tutor

- El API exige `parentalConsent: true` (o `acceptedTermsAndPrivacy: true`) al registrar cuenta de tutor
- La app móvil muestra aceptación de términos y privacidad antes del registro
