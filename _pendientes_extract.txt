Problema: Actualmente app.use(cors()) sin restricciones. Cursor dijo que era “fuera del alcance solo tests” pero es CRÍTICO de seguridad.
Implementar:
En src/config/env.ts agregar:

En .env.example agregar:
# CORS: lista separada por comas. En desarrollo dejar '*' o 'http://localhost:19006'# En producción usar dominios explícitos: 'https://eduplay.app,https://admin.eduplay.app'CORS_ALLOWED_ORIGINS=http://localhost:19006,http://localhost:3000
En src/app.ts REEMPLAZAR app.use(cors()) por:
import cors from 'cors';const corsOrigins = env.CORS_ALLOWED_ORIGINS;const corsOptions: cors.CorsOptions = {  origin: (origin, callback) => {    // Permitir sin origin (mobile apps, Postman, curl, server-to-server)    if (!origin) return callback(null, true);    // En desarrollo, permitir localhost    if (env.NODE_ENV === 'development') {      if (origin.startsWith('http://localhost:') || origin.startsWith('https://localhost:')) {        return callback(null, true);      }    }    // En producción, solo orígenes explícitos    if (corsOrigins === '*') {      return callback(null, true);    }    const allowedList = corsOrigins.split(',').map(o => o.trim()).filter(Boolean);    if (allowedList.includes(origin)) {      return callback(null, true);    }    callback(new Error(`CORS: origin ${origin} not allowed`));  },  credentials: true,  methods: ['GET', 'POST', 'PUT', 'PATCH', 'DELETE', 'OPTIONS'], allowedHeaders: ['Content-Type', 'Authorization', 'X-Requested-With'],  maxAge: 86400, // 24 horas de cache para preflight};app.use(cors(corsOptions));
Tests en tests/security.integration.test.ts:
Origen en allowlist → 200 + headers CORS correctos
Origen NO en allowlist → bloqueado (error CORS, no llega al handler)
Sin origin (mobile app) → permitido
OPTIONS /api/health con origin válido → 204 + headers preflight
Cambiar CORS_ALLOWED_ORIGINS a dominio específico y verificar rechazo de localhost
En desarrollo: npm run dev debe seguir funcionando sin problemas con Expo/web.
Entregable: CORS seguro con allowlist + tests + documentación en README.
